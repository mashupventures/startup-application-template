# LLM-friendly Startup Application Template

이 저장소는 스타트업이 자신의 회사와 제품을 **간결하고 구조적으로 설명할 수 있도록 돕는 텍스트 템플릿**을 제공합니다.

많은 스타트업 지원서는 보통 Pitch Deck(PDF) 형태로 제출됩니다.  
하지만 회사의 핵심 내용을 **구조화된 텍스트 형태로 정리하면**, 투자 검토 과정에서 더 빠르고 명확하게 이해하는 데 도움이 될 수 있습니다.

이 템플릿은 스타트업이 **문제, 제품, 시장, 팀, 성과 등 핵심 정보를 정리**할 수 있도록 만든 간단한 예시입니다.

또한 이러한 형식은 투자 검토 과정에서 **LLM을 활용한 요약 및 투자 메모 작성**을 보다 쉽게 할 수 있도록 돕습니다.

이 템플릿의 사용은 **완전히 선택 사항**입니다.  
기존의 Pitch Deck이나 사업계획서를 제출하셔도 전혀 문제 없습니다.

---

## 사용 방법

1. `sample.txt` 파일을 열어주세요
2. 템플릿 내용을 복사합니다
3. 각 항목에 맞게 스타트업 정보를 작성합니다
4. `.txt` 파일로 저장합니다
5. 투자 지원 폼에 업로드합니다

템플릿 구조를 참고해서 **자유로운 형식으로 작성하셔도 괜찮습니다.**

## 어떤 내용을 작성하면 좋을까요?

템플릿은 보통 투자 검토에서 중요하게 보는 내용을 중심으로 구성되어 있습니다.

예를 들어 다음과 같은 내용입니다.

- 회사 한 줄 소개
- 해결하려는 문제
- 제품 또는 서비스
- 왜 지금 이 문제를 해결하는 것이 중요한지
- 고객과 시장
- 현재까지의 성과
- 비즈니스 모델
- 경쟁 환경
- 창업팀 소개
- 투자 계획

꼭 긴 문서를 작성할 필요는 없습니다.
**핵심 내용을 간결하게 설명하는 것이 가장 좋습니다.**

---

## 왜 이런 형식을 사용하나요?

구조화된 텍스트 형태의 자료는 회사의 핵심 내용을 빠르게 이해하는 데 도움이 됩니다.
또한 검토 과정에서 중요한 정보를 정리하고 요약하는 데에도 유용합니다.
